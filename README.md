## Local Development

### 🧑‍🤝‍🧑 Clone the project

```sh
https://github.com/joysofcode/svelte-matching-game
```

### 📦️ Install dependencies

```sh
pnpm i
```

### 💿️ Run the development server

```sh
pnpm run dev
```
